# 코스닥 포지션 사이징 계산기

## 핵심 원칙

```
한 종목에서 최대 손실 = 전체 계좌의 2%
→ 이 원칙 하나가 계좌를 지킨다
```

---

## 입력값

| 항목 | 예시 |
|------|------|
| 총 계좌 금액 | 10,000,000원 |
| 종목당 최대 손실 비율 | 2% (기본값) |
| 진입 가격 | 7,200원 |
| 손절 가격 | 6,700원 |

---

## 계산 공식

```
최대 손실 금액 = 계좌 × 손실비율
             = 10,000,000 × 0.02 = 200,000원

주당 손실 = 진입가 - 손절가
          = 7,200 - 6,700 = 500원

매수 수량 = 최대 손실 금액 ÷ 주당 손실
          = 200,000 ÷ 500 = 400주

총 투자금 = 400 × 7,200 = 2,880,000원
계좌 비중 = 2,880,000 / 10,000,000 = 28.8%
```

---

## 등급별 사이징 조정

| 종목 등급 | 기본 비율 | 계좌 비중 한도 |
|---------|---------|------------|
| S급 (극저평가 + 거래량 확인) | 2% | 최대 30% |
| A급 (저평가 우량주) | 1.5% | 최대 20% |
| B급 (모멘텀/테마) | 1% | 최대 15% |
| 신규 테마 (검증 전) | 0.5% | 최대 10% |

---

## 코스닥 특수 규칙

```
1. 일일 변동폭 ±30% 제한
   → 소형주는 하루 만에 -30% 가능
   → 손절가를 진입가의 -7% 이내로 설정

2. 거래대금 < 50억인 종목
   → 매수 수량을 절반으로 줄이기 (유동성 리스크)

3. 동일 테마 2개 이상 보유 시
   → 테마 합산 비중 40% 초과 금지

4. 급등 당일 추격 매수
   → 포지션 크기 50% 축소
```

---

## 빠른 계산 템플릿

사용자가 진입가와 손절가를 주면 즉시 계산:

```
입력:
  계좌: [금액]원
  종목: [이름] ([코드])
  진입가: [원]
  손절가: [원]
  등급: [S/A/B]

출력:
  최대 손실 허용: [금액]원 (계좌의 X%)
  매수 수량: [수량]주
  총 투자금: [금액]원 (계좌의 XX%)
  목표가 +10%: [금액]원 → 수익 [금액]원
  목표가 +20%: [금액]원 → 수익 [금액]원
  리스크/리워드: 1 : X.X
```
