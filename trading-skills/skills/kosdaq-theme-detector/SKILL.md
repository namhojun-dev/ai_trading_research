# 코스닥 테마 탐지기

## 테마 생명주기 4단계

```
1단계 [태동]   : 소수 종목만 반응, 거래량 적음 → 선점 기회
2단계 [확산]   : 대장주 급등 + 유사 업종 동반 상승 → 최적 진입
3단계 [과열]   : 연일 상한가, 뉴스 도배, 개인 몰림 → 분할 익절
4단계 [소멸]   : 거래량 급감, 대장주 하락 전환 → 즉시 탈출
```

## Step 1 — 오늘 테마 확인

웹 검색으로 확인:
```
검색어: "코스닥 오늘 테마 급등 [날짜]"
확인처: 인포스탁, 콕스톡, CBC뉴스
```

---

## Step 2 — 테마 강도 평가

| 지표 | 강한 테마 | 약한 테마 |
|------|---------|---------|
| 상한가 종목 수 | 3개 이상 | 1개 이하 |
| 거래대금 집중도 | 테마 상위 5개 합산 5000억+ | 1000억 미만 |
| 지속 일수 | 3일 이상 연속 | 당일만 |
| 뉴스 촉매 | 실적/수출계약/정책 | 루머/기대감만 |

---

## Step 3 — 테마 소멸 신호 체크

다음 중 2개 이상 해당 시 탈출 검토:

```
□ 대장주가 전날 대비 하락 전환
□ 거래대금이 전날 대비 -40% 이상 감소
□ 뉴스가 반복되거나 더 이상 새 촉매 없음
□ 개인 매수 급증 + 기관/외국인 매도
□ 갭상승 후 음봉 마감 2일 연속
```

---

## Step 4 — 테마 추천 종목 매핑

테마 확인 후 kosdaq-value-screener와 교차:

```
우선순위: 활성 테마 + S/A급 저평가 = 최고 조합

예시:
  AI반도체 테마 활성 + 솔브레인홀딩스(PER 1.60) → S+급 진입 검토
  방산 테마 활성 + 코스닥 방산 소형주 → A급 진입 검토
```
