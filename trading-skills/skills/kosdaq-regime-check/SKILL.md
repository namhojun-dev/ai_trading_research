# 코스닥 시장 국면 체크 (Daily Regime Check)

## 언제 사용하나
- 매일 아침 장 시작 30분 전 (8:30~9:00)
- "오늘 시장 어때?" 질문 시
- 테마/종목 추천 전 선행 단계로 항상 실행

---

## Step 1 — 지수 방향 판단

다음을 확인하고 국면을 결정한다:

| 확인 항목 | 출처 |
|-----------|------|
| 코스닥 전일 종가 / 등락률 | investing.com/indices/kosdaq |
| 코스피 전일 종가 / 등락률 | investing.com/indices/kospi |
| 나스닥 전일 종가 / 등락률 | 미국 시장 선행 지표 |
| 코스닥 5일선 위/아래 | 단기 추세 |
| 코스닥 20일선 위/아래 | 중기 추세 |

### 국면 판정표

| 국면 | 조건 | 전략 |
|------|------|------|
| **강한 상승** | 코스닥 +1% 이상, 나스닥 전날 +1% 이상 | 공격 매수, 거래대금 상위 모멘텀 추격 가능 |
| **약한 상승** | 코스닥 +0~1%, 혼조 | 저평가 가치주 중심, 소량 진입 |
| **횡보** | ±0.5% 이내 | 관망 또는 저점 분할 매수 |
| **약한 하락** | 코스닥 -1~0% | 현금 비중 확대, 손절 우선 |
| **강한 하락** | 코스닥 -1% 이상 | 신규 진입 금지, 기존 포지션 재검토 |

---

## Step 2 — 외국인 수급 체크

```
확인: 외국인 순매수/순매도 (코스닥 기준)

연속 매도 3일 이상 → 하락 압력 지속 가능성
연속 매수 전환 → 반등 신호
```

| 수급 신호 | 해석 |
|-----------|------|
| 외국인 순매수 전환 | 강한 반등 신호 |
| 외국인 8일 이상 연속 매도 | 저점 근접 가능성 (역발상 매수 검토) |
| 기관+외국인 동반 매수 | 가장 강한 상승 신호 |

---

## Step 3 — 거래대금 수준

| 거래대금 | 의미 | 전략 |
|---------|------|------|
| 코스닥 전체 15조+ | 활황, 테마 강하게 살아있음 | 모멘텀 추격 가능 |
| 코스닥 전체 8~15조 | 보통 | 선별적 접근 |
| 코스닥 전체 8조 미만 | 관망장 | 소형주 유동성 주의 |

---

## Step 4 — 국면 판정 출력

아래 형식으로 최종 요약 출력:

```
=== 코스닥 시장 국면 (날짜) ===
지수:      코스닥 XXX.XX (전일 대비 X.XX%)
나스닥:    전일 +X.XX% (위험자산 선호 ON/OFF)
국면:      [강한상승 / 약한상승 / 횡보 / 약한하락 / 강한하락]
외국인:    [순매수 / 순매도] N일 연속
거래대금:  코스닥 XXX조 (활황/보통/관망)

오늘 전략 방향:
  - [구체적 한 줄 전략]
  - 진입 가능 여부: [YES / NO / 소량만]
```
