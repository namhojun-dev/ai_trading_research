# 코스닥 저평가 스윙 스크리너

## 필터 조건 (전부 충족해야 통과)

| 조건 | 기준값 | 이유 |
|------|--------|------|
| EPS | > 0 | 흑자 기업만 (적자 제외) |
| PER | < 10 | 이익 대비 저평가 |
| PBR | < 1.0 | 자산가치 이하 거래 |
| 시총 | > 300억 KRW | 최소 유동성 확보 |
| 거래대금 | > 30억/일 평균 | 단타/스윙 진출입 가능 |
| 부채비율 | < 150% | 재무 안전성 |

---

## 평가 등급 (ROE 추가 가중치)

| 등급 | 조건 | 의미 |
|------|------|------|
| **S급** | PER<3 + PBR<0.5 + ROE>20% | 극저평가 우량주, 최우선 |
| **A급** | PER<5 + PBR<0.7 + ROE>10% | 저평가 우량주 |
| **B급** | PER<10 + PBR<1.0 + ROE>0% | 스윙 검토 가능 |

## Step 1 — 데이터 수집

확인할 출처 (무료):
- **judal.co.kr** → 저PER 정렬, 저PBR 정렬
- **fnguide.com/SVO2** → 개별 종목 재무지표 상세
- **alphasquare.co.kr** → 테마별 종목 + 재무 필터

---

## Step 2 — 거래대금 교차 확인

```
kokstock.com/stock/amount.asp → 코스닥 거래대금 상위 목록
→ 저평가 종목이 거래대금 상위에 있으면 S+급 신호
→ 거래대금 갑자기 급증 = 세력 진입 가능성
```

---

## Step 3 — 스윙 진입 타이밍 필터

재무 조건 통과 후 차트 조건 확인:

| 패턴 | 설명 | 진입 적합도 |
|------|------|------------|
| **눌림목** | 전고점에서 -10~20% 조정 후 거래량 감소 | ★★★ 최적 |
| **횡보 돌파** | N일 박스권 → 거래량 터지며 돌파 | ★★★ |
| **52주 신고가** | 신고가 돌파 + 거래량 2배 이상 | ★★ |
| **급락 후 반등** | 전날 -5% 이상 → 다음날 반등 시도 | ★ 위험, 소량만 |

---

```

[S급] 종목명 (코드)
  PER X.X | PBR X.XX | EPS X,XXX원 | ROE XX%
  시총 XXX억 | 거래대금 XXX억
  업종: XXX
  진입 근거: [눌림목/돌파/반등]
  목표가: XX,XXX원 (+X%)
  손절가: XX,XXX원 (-X%)
  리스크/리워드: 1 : X.X

[주의사항]
  - 시장 국면: [regime-check 결과 참조]
  - 진입 전 거래량 확인 필수
```
