# 진입 직전 체크리스트 (Swing Entry Checklist)

종목 선택 후 매수 버튼 누르기 전 실행. 2분이면 충분.

---

## 필수 체크 (하나라도 NO면 진입 보류)

```
□ 오늘 시장 국면이 "횡보" 이상인가?
□ 종목의 EPS가 양수(흑자)인가?
□ 손절가를 미리 정했는가?
□ 이 종목으로 최대 손실이 계좌의 2% 이하인가?
□ 진입가 대비 목표가가 최소 1:1.5 리스크/리워드인가?
□ 오늘 이미 2개 이상 진입하지 않았는가?
```

---

## 포지션 사이징 계산 (kosdaq-position-sizer)

```
계좌 금액:    ________원
종목:         ________ (코드)
진입가:       ________원
손절가:       ________원  (진입가의 -7% 이내 권장)
등급:         [S/A/B]

→ 매수 수량: ________주
→ 총 투자금: ________원 (계좌의 __%)
```

---

## 진입 후 관리 규칙

| 상황 | 행동 |
|------|------|
| 당일 -5% 이상 | 손절 실행, 이유 불문 |
| +10% 도달 | 절반 익절, 나머지 홀딩 |
| +20% 도달 | 전량 익절 검토 |
| 테마 소멸 신호 2개 이상 | 수익/손실 무관하게 청산 |
| 3일 이상 보유 + 수익 없음 | 청산 후 재판단 |

---

## 진입 후 일지 기록 (kosdaq-trade-journal)

```
"[종목명] [가격]원에 [수량]주 매수.
 이유: [테마/저평가/패턴]
 손절: [가격]원
 목표: [가격]원"
```
