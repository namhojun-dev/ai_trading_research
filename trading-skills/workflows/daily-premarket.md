# 장 시작 전 30분 루틴 (Daily Pre-Market Workflow)

실행 시간: 매일 오전 8:30~9:00 (장 시작 30분 전)

---

## 실행 순서

### Step 1 — 시장 국면 체크 (10분)
스킬: `kosdaq-regime-check`

```
확인:
  □ 코스닥/코스피 전일 등락률
  □ 미국 나스닥 전일 결과
  □ 외국인 수급 방향
  □ 오늘 국면 판정: [강한상승/약한상승/횡보/하락]
```

**국면이 "강한하락"이면 여기서 중단. 오늘 신규 진입 없음.**

---

### Step 2 — 주도 테마 확인 (10분)
스킬: `kosdaq-theme-detector`

```
확인:
  □ 어제 상한가 종목들의 테마 분류
  □ 오늘 장 시작 전 뉴스 촉매 유무
  □ 테마 생명주기 단계 판단
  □ 소멸 임박 테마 → 보유 중이면 청산 검토
```

---

### Step 3 — 후보 종목 도출 (10분)
스킬: `kosdaq-value-screener`

```
필터 적용:
  □ PER < 10, PBR < 1, EPS > 0
  □ 거래대금 > 30억
  □ 활성 테마와 교차 → 최우선 후보
  □ 차트 패턴 확인 (눌림목/돌파 여부)

오늘 관심 종목 3개 이내로 압축
```

---

### 장 시작 직전 체크리스트

```
□ 오늘 국면에 맞는 전략인가?
□ 손절가 설정했는가?
□ 포지션 사이징 계산했는가? (kosdaq-position-sizer)
□ 동일 테마 집중 비중 40% 이하인가?
□ 오늘 신규 진입 종목 최대 2개 제한
```

---

## 빠른 판단 매트릭스

| 국면 | 테마 활성 | 전략 |
|------|---------|------|
| 강한 상승 | O | 모멘텀 + 저평가 동시 공략 |
| 강한 상승 | X | 저평가 단독 진입 |
| 약한 상승 | O | 테마 대장 소량 + 저평가 |
| 횡보 | O | 저평가만, 테마 추격 금지 |
| 하락 | - | 신규 진입 금지 |
